    Blueprint Image Exporter

    Blueprint Image Exporter is an Unreal Engine 5 editor plugin for exporting high-quality PNG images from graph-based assets without relying on viewport screenshots.

    It is designed for documentation, reviews, tutorials, changelogs, and any workflow where you need clean captures of Unreal graph content that can be shared outside the editor.

    What it does

    • Exports graph images as PNG files.
    • Works from Unreal Editor tooling instead of external screen capture workflows.
    • Supports Blueprint-oriented editor workflows with batch export in mind.
    • Keeps output suitable for polished documentation and team communication.

    Info

    This documentation lives inside the plugin folder so the docs setup stays isolated from the rest of the repository.

    Highlights

    • Editor-only plugin architecture for Unreal Engine 5.
    • Source-compatible support starting at UE 5.0.0.
    • Built for graph capture workflows where readability matters.
    • Intended for professional documentation and asset review pipelines.

    Example

    The plugin is implemented as native Unreal Engine modules. A typical module declaration looks like this:

    class FBlueprintImageExporterModule : public IModuleInterface
{
public:
    virtual void StartupModule() override;
    virtual void ShutdownModule() override;
};

    Why use it

    If your team needs to publish Blueprint-related visuals to internal docs, release notes, or public knowledge bases, this plugin helps you produce repeatable image output that is easier to maintain than manual screenshots.
