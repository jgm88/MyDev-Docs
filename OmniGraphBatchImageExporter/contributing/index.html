    Contributing and repository layout

    This repository ships one plugin tree per Unreal minor version under Plugins/OmniGraph_UE_<major>.<minor>/OmniGraph/. Each tree has its own OmniGraph.uplugin with a matching EngineVersion (for example 5.5.0 for UE 5.5).

    Supported engines

    Pick the folder that matches the engine you build against. Differences between UE minors are isolated in BlueprintImageExporterEngineCompat.h, BlueprintImageExporterCoreEngineCompat.h, and related sources inside that tree.

    The .uplugin lists EditorScriptingUtilities, Niagara, RigVM, ControlRig, and Metasound so plugin load order matches BlueprintImageExporterCore.Build.cs and BlueprintImageExporter.Build.cs module dependencies.

    Architecture

    Module Role
    BlueprintImageExporterCore Graph enumeration (UbergraphPages, FunctionGraphs, MacroGraphs), Slate capture, IImageWriteQueue PNG writes, UBlueprintImageExporterSettings.
    BlueprintImageExporter Toolbar tab, Project Settings (Plugins → OmniGraph), Content Browser context menu.

    Editor resolution is centralized in FBlueprintImageExporterEditorLocator so engine changes to asset editors can be adapted in one place.

    Local development

    • Use Tools/New-OmniGraphHarness.ps1 in the repository root to create minimal harness projects under Unreal Projects with a junction from Plugins/OmniGraph into the matching versioned tree in this repo.
    • Use Tools/Build-OmniGraphAll.ps1 to run BuildPlugin for each installed engine and produce Fab-ready zip archives.

    Usage (reminder)

    1. Edit → Plugins: enable OmniGraph (and dependency Editor Scripting Utilities if needed).
    2. Project Settings → Plugins → OmniGraph: resolution scale, bounds padding, grid/solid background, default export folder, filename pattern ({Asset}, {Graph}).
    3. Content Browser: right-click one or more Blueprint assets → Export Blueprint Graphs to PNG... and pick a folder.
    4. Optional: main editor toolbar button opens the settings tab.

    Notes

    • Export opens or reuses the Blueprint editor and drives SGraphEditor / SGraphPanel (zoom, bounds, JumpToRect) before rasterizing.
    • Hide grid: when disabled in settings, the capture temporarily swaps UEditorStyleSettings::GraphBackgroundBrush to a solid tint (restored after the scoped capture path).
